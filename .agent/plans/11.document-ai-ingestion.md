# 11. Document AI Ingestion Pipeline

## Complexity: 🔴 Complex

Replaces the synchronous OpenAI ingestion with an async Document AI pipeline. Touches: file upload endpoint, a new background worker, batch parsing with polling, chunk persistence, embedding, image extraction to Supabase Storage, Realtime status updates, and the file modal UI. Several moving parts that must compose correctly.

If a single execution pass fails partway, the system stays consistent because each step is idempotent at the row level (file row exists, chunks insert under file_id, status column drives the state machine).

---

## Goal

A new file uploaded by a user goes through:

```
upload → Supabase Storage
       → enqueue background job
       → Document AI batch parse (eu region)
       → for each chunk: persist row + embed + insert
       → for each figure chunk: extract image bytes → Storage → chunk_images row
       → mark project_file.status = 'ready'
       → Realtime push to UI at every state transition
```

After this plan: new uploads land in `document_chunks` and `chunk_images`. Old OpenAI ingestion code is **not yet removed** — it lives next to the new path so existing files keep working until plan 12 cuts the chat over.

## Architectural decisions

- **Online vs batch:** Layout Parser online cap is 15 pages; we always use **batch**. Batch is async, signed-URL-driven, with `LongRunningOperation` polling.
- **Storage path for originals:** `project-files/{user_id}/{file_id}/{filename}` in a new private Supabase Storage bucket `project-files`. Document AI reads via signed URL (15-min TTL) or via a temporary GCS staging bucket — we'll use **GCS staging** because Document AI's batch input expects `gs://` URIs.
- **GCS staging bucket:** `gs://sleek-rag-staging-{env}` in `eu` region. Files copied there at upload time, deleted after parsing succeeds.
- **Background worker:** FastAPI `BackgroundTasks` is **not enough** (it dies with the worker process). We use a simple loop: a `asyncio.create_task` worker started at app boot that polls a `pending_ingest_jobs` queue table. Single-process, single-replica acceptable for v1 — note this for scale later.
- **Embedding batching:** call `embeddings.create()` in batches of 100 chunks. Most files generate 50–500 chunks; one or two API calls per file.
- **Failure handling:** any step failure sets `status='failed'` + `ingest_error`, partial chunks are deleted (cascade via `file_id`), file stays in Storage for retry.

### 11.5 — The ingestion worker

Single-task event loop, started in `main.py` lifespan. Pseudocode-shaped real code:

```python
import asyncio
import io
import logging
from google.cloud import documentai_v1 as documentai
from google.cloud import storage as gcs
from langsmith import traceable

from app.config import settings
from app.db import supabase
from app.documentai_client import documentai_client, processor_name
from app.gemini_client import gemini_client

log = logging.getLogger(__name__)

POLL_INTERVAL_SEC = 3
DOCAI_POLL_INTERVAL_SEC = 5
MAX_DOCAI_WAIT_SEC = 30 * 60       # 30 min hard cap
EMBED_BATCH_SIZE = 100

_gcs_client = None
def _gcs():
    global _gcs_client
    if _gcs_client is None:
        from google.oauth2 import service_account
        creds = service_account.Credentials.from_service_account_file(
            settings.gcp_service_account_json_path
        )
        _gcs_client = gcs.Client(project=settings.gcp_project_id, credentials=creds)
    return _gcs_client


async def run_worker():
    """Main loop: pick next queued job, run it. One job at a time (v1)."""
    log.info("ingest worker started")
    while True:
        try:
            job = _claim_next_job()
            if job is None:
                await asyncio.sleep(POLL_INTERVAL_SEC)
                continue
            await asyncio.to_thread(_process_job, job)
        except Exception as exc:
            log.exception("worker tick failed: %s", exc)
            await asyncio.sleep(POLL_INTERVAL_SEC)


def _claim_next_job() -> dict | None:
    # Atomic claim via update with state='queued' filter.
    res = (
        supabase().rpc("claim_next_ingest_job").execute()
    )  # see migration 0004 for the SQL function
    return (res.data or [None])[0]


@traceable(run_type="chain", name="ingest.process_job")
def _process_job(job: dict):
    file_id = job["file_id"]
    try:
        # 1. Stage to GCS.
        gcs_in = _stage_to_gcs(job)
        supabase().table("ingest_jobs").update({"gcs_input_uri": gcs_in}).eq("id", job["id"]).execute()

        # 2. Run Document AI batch.
        document = _run_layout_parser(gcs_in)
        _set_status(file_id, "embedding")
        _patch_file(file_id, page_count=len(document.pages))

        # 3. Persist chunks + figures + embeddings.
        chunk_count = _persist_chunks(job, document)
        _patch_file(file_id, chunk_count=chunk_count)

        # 4. Done.
        _set_status(file_id, "ready")
        supabase().table("ingest_jobs").update({"state": "done", "finished_at": "now()"}).eq("id", job["id"]).execute()

        # 5. Cleanup staging.
        _delete_gcs(gcs_in)
    except Exception as exc:
        log.exception("ingest failed for file %s", file_id)
        supabase().table("project_files").update({
            "status": "failed", "ingest_error": str(exc)[:500]
        }).eq("id", file_id).execute()
        supabase().table("ingest_jobs").update({
            "state": "failed", "last_error": str(exc)[:500], "finished_at": "now()"
        }).eq("id", job["id"]).execute()
```

**Sub-functions** (same file):

```python
def _stage_to_gcs(job: dict) -> str:
    file_row = supabase().table("project_files").select("gcs_blob_path,filename,mime_type").eq("id", job["file_id"]).single().execute().data
    blob_bytes = supabase().storage.from_("project-files").download(file_row["gcs_blob_path"])
    bucket = _gcs().bucket(settings.gcs_staging_bucket)
    obj_name = f"ingest/{job['id']}/{file_row['filename']}"
    blob = bucket.blob(obj_name)
    blob.upload_from_string(blob_bytes, content_type=file_row["mime_type"])
    return f"gs://{settings.gcs_staging_bucket}/{obj_name}"


@traceable(run_type="retriever", name="documentai.layout_parser")
def _run_layout_parser(gcs_input_uri: str):
    """Sync online process for ≤15 pages, batch otherwise."""
    # For v1, always use the synchronous endpoint when file is small;
    # batch_process_documents otherwise. We detect by attempting the sync
    # call first and falling back to batch on InvalidArgument.
    bucket_name, obj_name = gcs_input_uri.replace("gs://","").split("/",1)
    raw_bytes = _gcs().bucket(bucket_name).blob(obj_name).download_as_bytes()
    process_options = documentai.ProcessOptions(
        layout_config=documentai.ProcessOptions.LayoutConfig(
            chunking_config=documentai.ProcessOptions.LayoutConfig.ChunkingConfig(
                chunk_size=500, include_ancestor_headings=True,
            )
        )
    )
    try:
        resp = documentai_client().process_document(request=documentai.ProcessRequest(
            name=processor_name(),
            raw_document=documentai.RawDocument(content=raw_bytes, mime_type="application/pdf"),
            process_options=process_options,
        ))
        return resp.document
    except Exception as e:
        if "exceeds" in str(e).lower() or "too large" in str(e).lower():
            return _run_batch(gcs_input_uri, process_options)
        raise


def _run_batch(gcs_input_uri: str, process_options):
    output_uri = f"gs://{settings.gcs_staging_bucket}/output/{gcs_input_uri.split('/')[-2]}/"
    op = documentai_client().batch_process_documents(request=documentai.BatchProcessRequest(
        name=processor_name(),
        input_documents=documentai.BatchDocumentsInputConfig(
            gcs_documents=documentai.GcsDocuments(documents=[
                documentai.GcsDocument(gcs_uri=gcs_input_uri, mime_type="application/pdf")
            ])
        ),
        document_output_config=documentai.DocumentOutputConfig(
            gcs_output_config=documentai.DocumentOutputConfig.GcsOutputConfig(gcs_uri=output_uri)
        ),
        process_options=process_options,
    ))
    op.result(timeout=MAX_DOCAI_WAIT_SEC)
    # Read the single output JSON from GCS and parse into a Document message.
    bucket_name = output_uri.replace("gs://","").split("/",1)[0]
    prefix = output_uri.replace(f"gs://{bucket_name}/","")
    for blob in _gcs().bucket(bucket_name).list_blobs(prefix=prefix):
        if blob.name.endswith(".json"):
            return documentai.Document.from_json(blob.download_as_bytes())
    raise RuntimeError("batch parse produced no output JSON")


def _persist_chunks(job: dict, document) -> int:
    file_id = job["file_id"]
    user_id = job["user_id"]
    project_id = supabase().table("project_files").select("project_id").eq("id", file_id).single().execute().data["project_id"]

    chunks = list(document.chunked_document.chunks)
    if not chunks:
        return 0

    # 1. Embed in batches.
    texts = [c.content for c in chunks]
    embeddings: list[list[float]] = []
    for i in range(0, len(texts), EMBED_BATCH_SIZE):
        batch = texts[i:i+EMBED_BATCH_SIZE]
        resp = gemini_client().embeddings.create(
            model=settings.gemini_embedding_model, input=batch,
            dimensions=settings.gemini_embedding_dim,
        )
        embeddings.extend(d.embedding for d in resp.data)

    # 2. Build chunk rows.
    chunk_rows = []
    figure_rows: list[tuple[int, bytes, str | None]] = []  # (chunk_index, image_bytes, mime)
    for idx, (chunk, emb) in enumerate(zip(chunks, embeddings)):
        block_type = _block_type(chunk)
        page_start = chunk.page_span.page_start if chunk.page_span else 1
        page_end = chunk.page_span.page_end if chunk.page_span else page_start
        heading_path = list(chunk.page_headers) if hasattr(chunk, "page_headers") else None
        figure_label = _extract_figure_label(chunk)
        chunk_rows.append({
            "file_id": file_id, "project_id": project_id, "user_id": user_id,
            "chunk_index": idx, "block_type": block_type, "content": chunk.content,
            "page_start": page_start, "page_end": page_end,
            "heading_path": heading_path, "figure_label": figure_label,
            "embedding": emb,
        })
        if block_type == "figure" and chunk.images:
            img = chunk.images[0]
            figure_rows.append((idx, img.image, img.mime_type or "image/png"))

    inserted = supabase().table("document_chunks").insert(chunk_rows).execute()
    inserted_ids = [r["id"] for r in inserted.data]

    # 3. Persist figure images.
    for chunk_idx, img_bytes, mime in figure_rows:
        chunk_id = inserted_ids[chunk_idx]
        path = f"{user_id}/{file_id}/{chunk_id}.png"
        supabase().storage.from_("chunk-images").upload(path, img_bytes, {"content-type": mime})
        supabase().table("chunk_images").insert({
            "chunk_id": chunk_id, "user_id": user_id,
            "storage_path": path, "caption": chunks[chunk_idx].content,
            "byte_size": len(img_bytes),
        }).execute()

    return len(chunk_rows)


def _block_type(chunk) -> str:
    # Layout Parser exposes block-level info on the chunk; map to our enum.
    # Default fallback is 'paragraph'. Heading detection via `page_headers` non-empty + short content.
    if chunk.images:
        return "figure"
    if hasattr(chunk, "block_type"):
        return chunk.block_type.lower()
    return "paragraph"


def _extract_figure_label(chunk) -> str | None:
    # Layout Parser captions usually start with "Figure X.Y" or "Abbildung X.Y".
    import re
    m = re.match(r"^\s*(Figure|Abbildung|Fig\.|Abb\.)\s*([\d.]+)", chunk.content or "")
    if m:
        return f"{m.group(1).rstrip('.')} {m.group(2)}"
    return None


def _set_status(file_id, status):
    supabase().table("project_files").update({"status": status}).eq("id", file_id).execute()


def _patch_file(file_id, **fields):
    supabase().table("project_files").update(fields).eq("id", file_id).execute()


def _delete_gcs(uri: str):
    bucket_name, obj_name = uri.replace("gs://","").split("/",1)
    _gcs().bucket(bucket_name).blob(obj_name).delete()
```

**Add the SQL claim function** in `0004_claim_job.sql`:
```sql
create or replace function public.claim_next_ingest_job()
returns setof public.ingest_jobs
language plpgsql security definer as $$
declare
  job public.ingest_jobs;
begin
  select * into job from public.ingest_jobs
    where state = 'queued'
    order by created_at
    for update skip locked
    limit 1;
  if not found then return; end if;
  update public.ingest_jobs
    set state = 'parsing', attempts = attempts + 1, started_at = now()
    where id = job.id
    returning * into job;
  return next job;
end;
$$;

revoke all on function public.claim_next_ingest_job() from public, anon, authenticated;
```
